    Система отчётов¶

    1. Цветовые зоны развития¶

    Каждый домен оценивается в процентах (0–100%) и попадает в одну из трёх зон.

    ⚠️ Фактические пороги из документа raw/berkana_iorr_v3_3_final_ru.docx (Таблица 70):

    Зона Диапазон Цвет severity Действие
    🟢 Зелёная ≥ 75% #22C55E typical Плановый мониторинг
    🟡 Жёлтая 40–74% #EAB308 monitor Стимуляция + повтор через 2 мес.
    🔴 Красная < 40% #EF4444 follow_up Консультация педиатра в ближайшее время

    Жёлтая зона начинается с 40% (не 50% как в исходном плане).

    В UI: каждый домен отображается как карточка с цветной плашкой и процентом. Критические домены (🔴) всегда отображаются первыми.

    3. PDF-отчёт¶

    Технология¶

    WeasyPrint (предпочтительно — HTML→PDF, поддерживает CSS, Unicode/кириллица) 

    pip install weasyprint

    Альтернатива: ReportLab (низкоуровневый, без CSS).

    Структура PDF (3–4 страницы)¶

    Страница 1: Обложка - Логотип Berkana Brain - Заголовок: "Отчёт о нейрофункциональном развитии" - Данные: имя ребёнка, дата рождения, возраст, дата скрининга - Использованный модуль - Дисклеймер жирным шрифтом - Водяной знак (диагональный, светло-серый)

    Страница 2: Сводный профиль - Таблица доменов с зонами и процентами - Общий уровень - Краткое описание каждого домена (2-3 предложения)

    Страница 3: Детальный анализ - Для каждого домена: zone badge + интерпретация + рекомендация - Ключевые маркеры, требующие внимания

    Страница 4: Рекомендации и следующие шаги - Рекомендации по доменам в 🔴/🟡 зонах - Дата следующего скрининга - Контакты платформы - Полный дисклеймер

    4. Backend — генерация отчёта¶

    backend/screening/report_generator.py¶

    def generate_report(session: ScreeningSession) -> Report:
    domain_scores = calculate_domain_scores(session)
    interpretations = get_interpretations(session.module, domain_scores)
    overall_level = get_overall_level(interpretations)
    delta_scores = calculate_delta(session)  # None если первый скрининг

    report = Report.objects.create(
        session=session,
        child=session.child,
        domain_scores=domain_scores,
        overall_level=overall_level,
        interpretations=interpretations,
        recommendations=build_recommendations(interpretations),
        delta_scores=delta_scores,
    )
    return report

    Алгоритм зон (фактические пороги из документа)¶

    ZONES = {
    'typical':   {'min': 75,  'max': 100, 'color': '#22C55E', 'label_ru': 'Зелёная зона'},
    'monitor':   {'min': 40,  'max': 74,  'color': '#EAB308', 'label_ru': 'Жёлтая зона'},
    'follow_up': {'min': 0,   'max': 39,  'color': '#EF4444', 'label_ru': 'Красная зона'},
}

def get_zone(score: float) -> str:
    if score >= 75: return 'typical'
    if score >= 40: return 'monitor'
    return 'follow_up'

def calculate_domain_score(answers_for_domain: list) -> float:
    """
    Формула из документа: сумма / макс_возможная × 100
    Вопросы с ответом 'not_tried' исключаются из знаменателя.
    """
    total_score = 0
    max_possible = 0
    for answer in answers_for_domain:
        if answer.value == 'not_tried':
            continue  # НЕ ПРОБОВАЛИ — исключается
        total_score += answer.score
        max_possible += 10  # макс балл за вопрос = 10
    if max_possible == 0:
        return 0.0
    return round((total_score / max_possible) * 100, 1)

    5. Опрос обратной связи (FeedbackSurvey)¶

    Экран /screening/{session_id}/feedback¶

    Появляется после complete_session, до перехода на страницу результатов. Пропускается нажатием "Пропустить — перейти к результатам".

    Вопросы: 1. "Насколько понятны были вопросы скрининга?" (scale 1–5) 2. "Был ли какой-то вопрос непонятным или некорректным?" (yes/no + text) 3. "Как вы оцениваете удобство интерфейса?" (scale 1–5) 4. "Ваши комментарии или предложения" (text, необязательно) 5. "Хотите ли вы, чтобы специалист проверил ваш отчёт?" (yes/no)

    API¶

    POST /api/sessions/{id}/feedback/
Body: {
  "answers": [
    {"question": "clarity", "value": "4"},
    {"question": "unclear_question", "value": "no"},
    {"question": "interface_rating", "value": "5"},
    {"question": "comment", "value": "Всё понятно"},
    {"question": "specialist_review", "value": "yes"}
  ]
}

    Модели¶

    class FeedbackSurvey(models.Model):
    session = models.OneToOneField(ScreeningSession, on_delete=models.CASCADE)
    submitted_at = models.DateTimeField(auto_now_add=True)
    is_skipped = models.BooleanField(default=False)
    wants_specialist_review = models.BooleanField(default=False)

class FeedbackAnswer(models.Model):
    survey = models.ForeignKey(FeedbackSurvey, on_delete=models.CASCADE, related_name='answers')
    question_key = models.CharField(max_length=50)  # 'clarity', 'interface_rating', etc.
    value = models.TextField()
