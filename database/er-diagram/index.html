    ER Diagram — Berkana Brain¶

    Полная диаграмма всех Django-моделей проекта (accounts, children, screening, specialists).

    erDiagram
    User ||--|| UserProfile : "profile"
    User ||--o| SpecialistProfile : "specialist profile"
    UserProfile ||--o{ Child : "children"
    UserProfile ||--o{ SpecialistAssignment : "assignments as parent"
    UserProfile ||--o{ ValidationAppointment : "appointments as parent"

    Child ||--o{ ScreeningSession : "sessions"
    Child ||--o{ Report : "reports"

    ScreeningModule ||--o{ Question : "questions"
    ScreeningModule ||--o{ ScoringRule : "scoring rules"
    ScreeningModule ||--o{ ScreeningSession : "sessions"

    ScreeningSession ||--o{ Answer : "answers"
    ScreeningSession ||--o| Report : "report"
    ScreeningSession ||--o| FeedbackSurvey : "feedback"

    Question ||--o{ Answer : "answered by"
    FeedbackSurvey ||--o{ FeedbackAnswer : "answers"

    Report ||--o{ SpecialistAssignment : "assigned to specialists"
    Report ||--o{ ValidationRecord : "validation records"
    Report ||--o{ ValidationAppointment : "appointments"

    SpecialistProfile ||--o{ SpecialistAssignment : "assignments"

    Домены скрининга¶

    Поле domain в Question, ScoringRule, Exercise:

    Код Название
    motor_gross Крупная моторика
    motor_fine Мелкая моторика
    speech Коммуникация и речь
    social Социально-эмоциональное
    self_care Самообслуживание и адаптивное поведение
    cognitive Когнитивное развитие
    sensory Сенсорная интеграция
    family Семья и анамнез
    red_flag Красные флаги (не участвует в скоринге)
    early_reg Ранние регуляторные маркеры (не участвует в скоринге)
