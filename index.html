    Обзор проекта Berkana Brain¶

    Что это?¶

    Berkana Brain — цифровая SaaS-платформа нейрофункционального скрининга развития детей. Система позволяет родителям самостоятельно проходить структурированные опросники онлайн и получать автоматически сформированный отчёт о профиле развития ребёнка.

    Важно: Система НЕ ставит медицинских диагнозов. Это инструмент наблюдения и скрининга.

    Проблема, которую решает¶

    Традиционные методы оценки развития ребёнка требуют: - Участия квалифицированных специалистов - Личного присутствия - Значительных временных и финансовых затрат

    Это ограничивает доступность инструментов раннего скрининга для большинства семей.

    Решение¶

    Цифровая платформа, где родители могут: 1. Самостоятельно проходить скрининговые опросники (дома, в удобное время) 2. Получать аналитические отчёты о развитии ребёнка 3. Отслеживать динамику развития в сравнении с предыдущими скринингами 4. Получать рекомендации по наблюдению

    Бизнес-цели¶

    # Цель Описание
    3.1 Доступность Родители проходят скрининг без специалиста
    3.2 Автоматизация Алгоритмический анализ ответов
    3.3 Поддержка специалистов Специалисты отслеживают динамику своих клиентов
    3.4 Структурированные отчёты Профиль развития + рекомендации
    3.5 SaaS-масштабируемость Мультиязычность, много пользователей, модульность

    Целевая аудитория¶

    Первичная: Родители / законные представители детей (0–18 лет)

    Вторичная: - Психологи и педагоги (анализ результатов, рекомендации) - Специалисты центров развития - Исследователи (агрегированные данные)

    Текущее состояние (23 мая 2026)¶

    Компонент Статус
    Лендинговая страница Готово
    Авторизация Telegram phone auth + JWT + выбор роли
    Кабинет родителя Профиль, дети, прохождение скрининга, отчёты, PDF, feedback
    Django backend Реализован: accounts, children, screening, specialists, notifications
    База данных Django models + PostgreSQL для рабочей среды; локально возможен sqlite при пустом DATABASE_URL
    Telegram Bot Реализован для auth-кодов и уведомлений по консультациям
    Скрининговые модули Импортируемые BERKANA-модули; основной рабочий контент 0–6 месяцев
    Кабинет специалиста Верификация, очередь, клиенты, консультации, inline-валидация
    Кабинет администратора Обзор, статистика, родители, специалисты, авто-назначение, фильтры для прозвона родителей без/с повторным скринингом
